## Vore System Configuration

## Maximum number of bellies allowed per layout (default: 10)
#VORE_MAX_BELLIES 10

## Maximum number of prey that can be eaten at once (default: 3)
#VORE_MAX_PREY 3

## Time in seconds it takes to eat someone (default: 4)
#VORE_DELAY 4

## Time in seconds it takes to eject someone (default: 2)
#VORE_EJECT_DELAY 2

## Maximum number of belly layout slots per character (default: 20)
#VORE_MAX_BELLY_LAYOUTS 20

## Maximum burn damage a belly can be set to deal (default: 2.5)
#VORE_MAX_BURN_DAMAGE 2.5

## Maximum brute damage a belly can be set to deal (default: 2.5)
#VORE_MAX_BRUTE_DAMAGE 2.5

## Minimum time in seconds that can be set for escaping a belly (default: 2)
#VORE_MIN_ESCAPE_TIME 2

## Default time in seconds for escaping a belly (default: 15)
#VORE_DEFAULT_ESCAPE_TIME 15

## Maximum time in seconds that can be set for escaping a belly (default: 60)
#VORE_MAX_ESCAPE_TIME 60

## Maximum length of custom belly verbs (default: 20)
#VORE_MAX_VERB_LENGTH 20

## Maximum length of vore messages (default: 160)
#VORE_MAX_MESSAGE_LENGTH 160

## Minimum length of vore messages (default: 10)
#VORE_MIN_MESSAGE_LENGTH 10

## If enabled, prevents people with prey inside them from being eaten
VORE_MATRYOSHKA_BANNED

## If enabled, automatically disables suit sensors when prey is eaten
VORE_DISABLES_SENSORS

## If enabled, prevents mobs in crit or death from engaging in vore
VORE_NO_DEAD

## If enabled, mobs with no player cannot be pred or prey
VORE_REQUIRES_PLAYER
